# Spatio-temporal models — Reaction-diffusion

> Modelling diffusion and front-propagation phenomena in mathematical biology.

**Context**: Modelling of living systems course (M1).

## What it does
- **Diffusion equation**: verification of the fundamental solution (Gaussian kernel) and numerical solving.
- **Reaction-diffusion equations**: simulation of **Fisher-KPP** and **Kolmogorov** travelling fronts, which model e.g. population or gene spread.
- Generation of **animations** of the spatio-temporal dynamics.

## Stack
Python (NumPy, Matplotlib), Jupyter.

## Files
- `TP.ipynb`, `TP_TD.ipynb` — notebooks
- `animation_fisher.mp4`, `animation_kolmogorof.mp4` — front animations

## Run
Open `TP.ipynb` in Jupyter.
