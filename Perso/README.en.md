# Personal projects — Games

> Small game projects for fun: a Connect 4 with AI and a card-game prototype.

**Context**: personal projects.

## Connect 4 with AI (`P4.py`)
A Connect 4 game playable against an AI:
- **Minimax algorithm** with several difficulty levels (depth 2 to 8)
- Board and win-probability display via Matplotlib
- Full alignment-detection logic

```bash
python P4.py
```

## Card-game prototype — Attack/Defense (`att_def/proto.py`)
Prototype of a turn-based card game (attack, defense, shields, face-down charge cards) with a **wxPython graphical interface**.

```bash
python att_def/proto.py
```

## Stack
Python (NumPy, Matplotlib, wxPython).
