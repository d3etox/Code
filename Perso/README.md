# Projets personnels — Jeux

> Petits projets de jeux pour le plaisir : un Puissance 4 avec IA et un prototype de jeu de cartes.

**Contexte** : projets personnels.

## Puissance 4 avec IA (`P4.py`)
Jeu de Puissance 4 jouable contre une IA :
- **Algorithme minimax** avec plusieurs niveaux de difficulté (profondeur 2 à 8)
- Plateau et affichage des probabilités via Matplotlib
- Logique complète de détection d'alignements

```bash
python P4.py
```

## Prototype de jeu de cartes — Attaque/Défense (`att_def/proto.py`)
Prototype d'un jeu de cartes tour par tour (attaque, défense, boucliers, cartes de charge face cachée) avec **interface graphique wxPython**.

```bash
python att_def/proto.py
```

## Stack
Python (NumPy, Matplotlib, wxPython).
